 ```

 ```

 **Use a virtual environment to avoid Global conflict**
 ```
 # Unix/Linux/macOS
 python3 -m venv .venv
 source .venv/bin/activate

 # Windwos (PowerShell)
 python3 -m venv .venv
 .\.venv\Scripts\Activate.ps1
 ```
